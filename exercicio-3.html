Lado cliente x Lado servido

Este é um otimo ponto para se debruçar em desenvolvimento Web. Nesse post vamos esclarecer melhor os "dois lados da moeda"

lados em transformação

Antigamente, nos primordios da web, era comum o lado cliente ficar com todo "trampo" rsrs. O lado servidor continham apenas a camada de dados. Mas ao longo dos anos esse jogo virou.

lados atualmente

Atualmente, o que era camada do cliente e camada de dados(servidor) se tornou camada cliente e camada de aplicação, dados e apresentação(servidor)

Tecnologias usadas em cada camada

lado cliente

html, css, javascript

lado servidor

php, bancos de dados e outras
